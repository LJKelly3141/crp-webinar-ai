# Optional Extension Prompts

These are the **original step-by-step prompts** from the April 2026 webinar,
preserved here for situations where the demo slot is longer than 15–18 minutes
or for use during Q&A.

The canonical condensed script in [`../prompts.md`](../prompts.md) collapses
each demo into a single combined C.O.R.E. prompt. These extensions break
Demo 1 back into four separate prompts (one per C.O.R.E. step) with multiple
clarifying follow-ups — adds ~15 minutes if run in full.

## When to use these

- **30+ minute demo slot** — present Demo 1 as 4 separate steps so the
  audience sees each C.O.R.E. piece called out explicitly.
- **Live Q&A** — when an attendee asks "but how do I dig deeper after
  the first response?" use the clarifying follow-ups here.
- **Workshop format** — pair attendees and have them work through the
  4-step version themselves.

## Files

| File | What it contains |
|------|------------------|
| `full-stepped-prompts.md` | Markdown copy-paste version of Demo 1 as 4 separate C.O.R.E. prompts, each with a clarifying follow-up |
| `full-stepped-demo-plan.txt` | Plain-text demo plan with the full 4-step version of Demo 1 plus all the original follow-up prompts (seasonality, summary stats, missing-value handling, etc.) |

## Notable bonus follow-ups available here

- "Provide summary statistics of the data set. Focus on the possibility
  of using this data to forecast cash inflow."
- "Replace observations of 0 that are actually missing with truly
  missing values. Store the result as an xlsx file."
- "You mention we have month due. Can we learn anything about
  seasonality of late payments?"
- "You mentioned employee count — is the relationship linear, or do
  mid-size companies behave differently than very small or very large ones?"
- "If I were presenting these findings to my collections team, what
  caveats should I include?"
